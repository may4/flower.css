# flower.css
引用方法
```html
<head>
  <link rel="stylesheet" href="flower.css">
</head>
```
或者
```html
<head>
  <link rel="stylesheet" href="http://null_217_4236.oschina.io/flower-css/flower.css">
</head>
```
最后你需要添加class名如下
```html
<a class="button inLeft" href="#">BUTTON</a>
<button class="button inUp">BUTTON</button>
```
* `inLeft`
* `inUp`
* `inRight`
* `inDown`
* `inupLeft`
* `inupRight`
* `indownLeft`
* `indownRight`
* `cloBetween`
* `cloAngle`
* `cloTangent`
* `cloX`
* `cloY`
* `Roll`
* `Ball`
* `Elastic`
* `Schedule`
* `Across`
* `AlternateX`
* `AlternateY`
* `Flower`
* `iconFlip`
* `iconRotate`
